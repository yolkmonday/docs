# Kirimi

## Solusi Praktis Kirim Notifikasi WhatsApp dengan Mudah!

Kirim pesan via API dan jadwalkan pengiriman pesan WhatsApp dengan Kirimi. Coba Gratis!

## Fitur Unggulan

### Kirim Pesan dengan API
Nikmati kemudahan pengiriman pesan dengan API Kirimi yang mudah digunakan.

### Pengiriman Pesan Massal
Kirimi memungkinkanmu mengirim pesan ke banyak kontak dengan cepat. Satu klik, semua pesan akan terkirim ke daftar kontak yang dipilih.

### Jadwalkan Pengiriman Pesan
Atur jadwal pesan untuk dikirim secara otomatis di waktu yang kamu tentukan.

### Auto Reply
Buat balasan otomatis untuk setiap pesan masuk. Sesuaikan balasannya agar sesuai dengan kebutuhanmu!

### Webhook
Dapatkan notifikasi pesan masuk secara langsung melalui webhook Kirimi.
